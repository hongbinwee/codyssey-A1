# AGENTS.md

## Workspace Overview

### 개요

이 작업공간은 Codyssey A 계열 미션을 폴더별로 분리해 관리하기 위한 루트 작업공간이다.

현재 진행 중인 작업은 `A1-1`이며, 해당 미션 산출물과 보조 자료는 `A1-1/` 폴더 안에서 관리한다.

GitHub에서도 같은 구조로 보이도록 각 미션의 산출물과 보조 자료는 해당 폴더 안에 모아 관리한다.

### 폴더 구조

- `A1-1/`
  - 현재 진행 중인 A1-1 미션 산출물과 보조 자료

## Purpose

이 작업공간은 Codyssey A 계열 미션을 폴더별로 나누어 관리하기 위한 공용 작업공간이다.

각각의 결과물은 미션 이름에 맞게 보관한다.

## Working Rules

1. 3단계 이상의 과정이 필요하거나 구조적 결정이 필요한 작업은 먼저 계획을 세운다.
2. 여러 파일을 동시에 만들거나 수정하는 작업은 먼저 Spec을 만든다.
3. 작업이 꼬이거나 기준이 흔들리면 즉시 멈추고 계획을 다시 세운다.
4. 미션별 산출물은 해당 미션 폴더 안에서 관리한다.
5. 루트 문서는 작업공간 안내와 현재 활성 미션 연결 역할만 담당한다.
6. 임의로 넘겨짚지 않는다. 모호하면 질문해 명확히 한다.
7. 필요한 미션 산출물에 맞게 산출물이 만들어졌는지, 필요한 부분은 없는지 주기적으로 체크한다.

## Planning And Spec

다음 작업은 바로 실행하지 말고 먼저 계획 또는 Spec을 만든다.

- 미션 산출물 전체 구조 설계
- 최종 패키지 설계
- 10턴 이상 대화 로그 시나리오 설계
- 여러 파일을 동시에 만들거나 수정하는 작업

계획/Spec에는 목표, 입력 자료, 작업 단계, 산출물, 검증 방법, 확인 질문을 포함한다.

## Communication

기본 언어는 한국어다.

답변은 미션 통과와 산출물 작성에 직접 도움이 되도록 한다. 핵심을 먼저 말하고, 추상적인 설명보다 실행 가능한 단계와 예시를 우선한다.
